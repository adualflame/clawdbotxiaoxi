# Meme Scanner API 配置指南

## 方案一：Bitquery API（推荐）

Bitquery 提供 Four.meme 和链上数据的 API，有免费额度。

### 步骤：

1. 访问 https://bitquery.io/
2. 点击 **Sign Up** 注册账号
3. 登录后进入 **IDE** 或 **Account**
4. 找到 **API Key**，复制给我

### 免费额度：
- 10,000 次/月

---

## 方案二：Dune Analytics

Dune 可以查询链上数据，包括新币发行。

### 步骤：

1. 访问 https://dune.com/
2. 注册账号
3. 进入 **Settings** → **API**
4. 生成 API Key，复制给我

---

## 方案三：DexScreener API（最简单）

DexScreener 有公开 API，可以获取新币和热门币。

### 无需注册，直接可用！

我可以直接用这个 API 获取：
- 最新上线代币
- 热门交易对
- 价格和流动性数据

---

## 建议

先用 **方案三 DexScreener**，无需注册，我现在就可以接入。

如果需要更详细的数据（Twitter 热度等），再注册 Bitquery。
