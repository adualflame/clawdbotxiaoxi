## 六、综合评级与操作建议

### 6.1 评级标准

| 等级 | 分数区间 | 含义 |
|------|----------|------|
| S | 90-100 | 极度低估，重仓机会 |
| A | 80-89 | 明显低估，标准仓位 |
| B | 70-79 | 轻度低估，小仓位 |
| C | 60-69 | 合理估值，观望 |
| D | 50-59 | 轻度高估，不建议 |
| F | <50 | 高估或高风险，回避 |

### 6.2 抄底区间计算

**公式：**
```
激进抄底价 = 当前价 × (1 - 预期跌幅 × 0.5)
标准抄底价 = 当前价 × (1 - 预期跌幅 × 0.7)
保守抄底价 = 当前价 × (1 - 预期跌幅 × 0.9)

预期跌幅 = f(解锁压力, 技术面, 市场情绪)
```

**解锁压力调整系数：**
- 近3月解锁 >30%：预期额外跌幅 +15%
- 近3月解锁 20-30%：预期额外跌幅 +10%
- 近3月解锁 10-20%：预期额外跌幅 +5%
- 近3月解锁 <10%：无调整

### 6.3 仓位建议

| 评级 | 建议仓位 | 入场策略 |
|------|----------|----------|
| S级 | 30-50% | 分3批建仓 |
| A级 | 20-30% | 分2批建仓 |
| B级 | 10-20% | 单次建仓 |
| C级 | 0-10% | 观望或极小仓位 |
| D/F | 0% | 不参与 |

## 七、时间维度预测

### 7.1 短期 (1-4周)

重点看：
- 技术面（支撑位、RSI）
- 短期催化剂
- 解锁事件

### 7.2 中期 (1-6月)

重点看：
- 估值分析
- 中期催化剂
- 赛道趋势

### 7.3 长期 (6月+)

重点看：
- 基本面
- 竞争格局
- 宏观周期
