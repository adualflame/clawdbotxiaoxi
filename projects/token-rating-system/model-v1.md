# 代币综合评估体系 v1.0

*创建日期: 2026-02-13*

## 设计原则

1. **数据必须实时获取** - 不用缓存，每次评估调 API
2. **多源交叉验证** - 至少 2 个数据源对比，偏差>5%标记异常
3. **量化评分** - 每个维度有明确的计算公式
4. **时间维度** - 分短期/中期/长期分别评估

## 七、抄底区间计算

**公式：**
```
基准跌幅 = (100 - 总评分) / 100 × 30%
解锁调整 = 近3月解锁比例 × 0.5
预期跌幅 = 基准跌幅 + 解锁调整

激进抄底价 = 当前价 × (1 - 预期跌幅 × 0.5)
标准抄底价 = 当前价 × (1 - 预期跌幅 × 0.8)
保守抄底价 = 当前价 × (1 - 预期跌幅 × 1.0)
```

**示例：**
- 总评分 75 分，近3月解锁 15%
- 基准跌幅 = 25/100 × 30% = 7.5%
- 解锁调整 = 15% × 0.5 = 7.5%
- 预期跌幅 = 15%
- 当前价 $1.00
- 激进：$0.925 / 标准：$0.88 / 保守：$0.85
