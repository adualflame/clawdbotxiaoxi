# 代币分析工作流

## 两种模式

### 1. 简洁模式（单纯查询）
用户只是问价格或简单查询时：
- 结论 + 关键风险点
- 不超过 5 行

### 2. 完整模式（涉及具体项目/投资决策）
输出完整评分报告，包含：
- 合约安全 (30分)
- 流动性 (25分)
- 持仓分布 (25分)
- 项目背景 (20分)

## 评分标准

详见：`memory/token-rating-system.md`

## 数据来源

- GoPlus API（合约安全）
- DexScreener（流动性、交易量）
- 链上数据（持仓分布）

## 自动触发
