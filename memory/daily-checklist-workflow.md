# 每日任务检查工作流

## 每天启动时

1. 读 `SOUL.md` + `USER.md` + `MEMORY.md`
2. 读 `memory/YYYY-MM-DD.md`（今天+昨天）
3. 检查待办完成情况
4. 根据任务类型读对应工作流文件

- 推特 → `twitter-workflow.md`

- PPT → `ppt-workflow.md`

## 每日结束时

- 重要决定写入 `MEMORY.md`
