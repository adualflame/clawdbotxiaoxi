## 📝 内容模板库

### 1. 市场观察类
```
📊 市场快报

BTC: $XX,XXX (24h: +X.X%)
恐惧指数: XX

观察：
- [核心观点]
- [数据支撑]

我的看法：[判断]

#BTC #加密货币
```

### 2. 链上数据类
```
🐋 巨鲸动向

过去24h:
- [交易所流入/流出]
- [大户地址变化]

解读：[意味着什么]
```

### 3. 热点快评类
```
🔥 热点速评

事件：[发生了什么]

影响：
- 短期：[XX]
- 长期：[XX]

我的判断：[观点]

你怎么看？👇
```

### 4. 项目分析Thread
```
🧵 [项目名] 深度拆解

1/ 最近很火，研究后分享：

2/ 📌 基本面
- 做什么/团队/融资

3/ 💰 代币经济
- 总量/流通/解锁

4/ ⚠️ 风险点

5/ 🎯 结论

NFA
```

### 5. 争议观点类
```
💬 可能会被骂，但我还是想说：

[反主流观点]

理由：
1. [论据1]
2. [论据2]

不同意？来辩👇
```

### 6. 个人复盘类
```
📈 本周复盘

✅ 做对了：
❌ 做错了：
💡 下周计划：

交易路上一起成长 🤝
```

## ⚠️ 执行规则

- **直接出推文**，不用先输出市场数据报告
- 小晰内部查数据，只呈现最终推文内容
- 保持简洁，ALEX 复制即可发布
- **数据必须查实**，不能凭感觉编造价格等数据

## 📱 朋友圈 vs 推特风格区别

**推特：**
- 数据向、专业分析
- 可带 #标签
- 面向公众传播

**朋友圈：**
- 活泼、幽默、有生活气息
- 引起好奇和讨论欲望
- 像跟朋友聊天，不是发公告
- 不带标签
- **不是搬运推特内容！要重新写**
