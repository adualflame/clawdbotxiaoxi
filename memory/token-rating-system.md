# 代币评分系统 (Token Rating System)

创建日期：2026-02-04

## 📋 评分维度

### 1. 合约安全 (30分)
- 合约是否开源验证 (10分)
- 是否有蜜罐/貔貅风险 (10分)
- 权限函数风险（mint、pause、blacklist等）(10分)

### 2. 流动性 (25分)
- 流动性池大小 (10分)
- LP是否锁仓/销毁 (10分)
- 买卖税率是否合理 (5分)

### 3. 持仓分布 (25分)
- 前10持仓集中度 (10分)
- 是否有巨鲸控盘风险 (10分)
- 持币地址数量 (5分)

### 4. 项目信息 (20分)
- 官网/白皮书 (5分)
- 社交媒体活跃度 (5分)
- 团队背景/KOL背书 (5分)
- 上线时间/交易历史 (5分)

---

## 📊 评级标准

| 分数 | 等级 | 风险提示 |
|------|------|----------|
| 85-100 | A | ✅ 低风险，可关注 |
| 70-84 | B | ⚠️ 中低风险，谨慎参与 |
| 50-69 | C | ⚠️ 中等风险，小仓试水 |
| 30-49 | D | 🚨 高风险，不建议参与 |
| 0-29 | F | ❌ 极高风险，大概率骗局 |

---

## 🔧 使用流程

1. 收到代币合约地址
2. 查询链上数据（DEXScreener、GoPlusLabs等）
3. 按维度打分
4. 输出评估报告
