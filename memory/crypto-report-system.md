# 加密信息报告体系

创建日期：2026-02-05

## ⏰ 推送节奏

| 时间 | 类型 | 内容重点 |
|-----|------|---------|
| 09:00 | 📊 早报 | 隔夜市场概览 + 重要新闻 |
| 15:00 | 🔥 Alpha | 链上热点 + 潜力机会 |
| 21:00 | 📈 晚报 | 当日总结 + 明日关注 |

## 📋 报告模板

### 📊 早报模板
```
📊 加密早报 | MM-DD

【市场概览】
BTC: $XX,XXX (24h: X%)
ETH: $X,XXX (24h: X%)
恐惧指数: XX

【隔夜要闻】
• [新闻1]
• [新闻2]

【今日关注】
• [事件/数据]
```

### 🔥 Alpha 模板
```
🔥 Alpha 速递 | MM-DD

【链上热点】
• [热门代币/项目]

【潜力机会】
🎯 项目: [名称]
📍 链: [BSC/SOL/ETH]
💰 合约: [地址]
⚠️ 风险: [高/中/低]
📝 理由: [简述]

【KOL 动态】
• [大V发言摘要]
```

### 📈 晚报模板
```
📈 加密晚报 | MM-DD

【今日回顾】
• BTC: $XX,XXX → $XX,XXX
• 涨幅榜: [TOP3]
• 跌幅榜: [TOP3]

【重要事件】
• [事件总结]

【明日关注】
• [预告/提醒]
```

## ⚠️ 筛选规则

- 只推有价值的信息，过滤噪音
- 避免重复推送相同项目
- Alpha 需有明确理由和风险提示
- 宁缺毋滥，没有好机会就不硬凑
