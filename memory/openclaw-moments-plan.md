# OpenClaw 朋友圈种草计划

创建日期：2026-02-05
周期：30天（2月5日 - 3月6日）
目标：通过朋友圈科普，让朋友了解 OpenClaw 的价值

## 📌 定位

- **风格**：生动、幽默、风趣，像跟朋友聊天
- **视角**：真实用户体验，不是硬广
- **分工**：小晰写文案 → ALEX 发布

## 📅 四周主题规划

| 周 | 主题 | 内容方向 |
|---|------|---------|
| W1 | 初识篇 | 这是什么？能干嘛？第一次对话 |
| W2 | 日常篇 | 怎么用起来的？真实场景 |
| W3 | 进阶篇 | 技巧、自动化、骚操作 |
| W4 | 感悟篇 | 人机协作的思考、未来展望 |

## ⚠️ 写作原则

- 每条带一个 hook（引发好奇）
- 不喊口号，讲真实故事
- 适当自嘲和幽默
- 控制长度，适合朋友圈阅读
